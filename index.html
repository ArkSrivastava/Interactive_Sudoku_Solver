Interactive Sudoku Solver

How to Use

Enter numbers (1-9) in the cells where you know the values.

Leave cells empty where you want the solver to find values.

Click "Solve Puzzle" to find the complete solution.

Use "Reset Grid" to start over with a blank grid.

Click "Try Example" to load a sample puzzle.

Navigate between cells using arrow keys or tab.
